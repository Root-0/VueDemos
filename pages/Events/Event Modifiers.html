Listen to events with v-on:event_name
Assign a method on event that takes place.
Example: < button v-on:mousemove="show_co-ordinates" > Hide details < / button >
Example with event modifier: < button v-on:mousemove.stop = "" > Hide details < / button >

Use event modifiers on an event to stop propagation or preventdefault.
Modifiers:
.stop = stop event propagation
.prevent = for form prevent default

Example: Show mouse X and Y co-ordinates on hover-event of following div. Stop event inside dead area.

Mouse co-ordinates: X = {{x}} and Y = {{y}}
Dead area (stop event)

                
                    < button v-on:mouseover="update_coordinates"> Show < / button >
                    
                    < div v-on:mouseover.stop=""> Dead area < / div >
                    
                    ~~~ Javascript ~~~ 
                    methods: {
                        update_coordinates: function(){
                            this.x = event.clientX;
                            this.y = event.clientY;
                        }
                    }
