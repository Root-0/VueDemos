Listen to events with v-on:event_name
Assign a method on event that takes place.
Example: < button v-on:click="hide_section" > Hide details < / button >

You can pass arguments with events inside methods.
Example: < button v-on:click="increase_counter(5)" >Increase by 5< / button >

Example: Change value of counter on click-event of following buttons.

Increase Decrease
Counter: {{counter}} 

                
                    < button v-on:click="increase_counter"> Increase < / button >
                    < button v-on:click="increase_counter(5)"> Increase by 5 < / button >

                    methods: {
