We can specify a property to attach instead of condition/classes inside :class
Whenever the property changes, classes get updated.
< div v-bind:class="property / method / computed property" >< / div >
The above syntax means the data inside property or the return value of method will be added as class.

You can also use array syntax for class/style binding
< div class="p-3 text-white" :style = "[ method()/property, { width: '150px', borderWidth: '5px', borderStyle: 'solid'}, { height: '150px'}]" >
