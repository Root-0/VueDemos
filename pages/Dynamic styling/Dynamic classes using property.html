We can specify a property to attach instead of condition/classes inside :class
Whenever the property changes, classes get updated. < div v-bind:class="property / method / computed property" >< / div >
The above syntax means the data inside property or the return value of method will be added as class.

Example: Change color of box by specifying the color class inside input.

demo box

Copy 
                    
                        < input type="text" v-model = 'bg_color' >
                        
                        < div class="p-3" :class = "bg_color" >
                            demo box
                        < / div >

                        ~~~ Javascript ~~~ 
                        data: {
                            bg_color: 'bg-white'
                        }
