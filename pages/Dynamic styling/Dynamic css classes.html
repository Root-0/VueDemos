You can conditinally add/remove classes with vuejs. We can pass an object to v-bind:class to dynamically toggle classes:
< div v-bind:class="{ classname: property_truthiness }" >< / div >
The above syntax means the presence of the active class will be determined by the truthiness of the data property.

You can have multiple classes toggled by having more fields in the object.
< div v-bind:class="{ class_1 : property_1, class_2 : property_2, class3 : property_3 }" >< / div >

Example: Toggle cardproperty on button click & change class of demo-box depending on property value.
