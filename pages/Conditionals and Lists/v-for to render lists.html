Loop through an array or object using v-for
Use v-for on the element you wish to duplicate
< li v-for = 'element in array' > { { element.name } }: { { element.description } } < / li >

Example: Loop through array & display its contents.

  • Planets

Copy 
                    
                        < ul >
                            < li v-for = '(planet,index) in planets'>{ { index + 1 } }  { { planet } } < / li >

                            < !-- print key value pair of object -- >
                            < li v-for = 'user in users'>
                                < div v-for="(myvalue, mykey) in users">
                                    {{ mykey }} : {{ myvalue }}
                                < / div >
                            < / li >
                        < / ul >
