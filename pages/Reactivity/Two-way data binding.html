v-model directive tells vuejs to setup a 2-way data binding for this element and the property you want to bind in both directions. Changing property via v-model will update the property everywhere it is used in your app.

Example: Bind car price to input & on change of car price, the discounted price should change.

Car price: {{carprice}}
10% Discounted price : {{ carprice - (carprice*0.10) }}

Copy 
                    
                        < input type="text"  v-model = 'carprice' >
                        
                        car price: { { carprice } }
                        10% Discounted price : { { carprice - (carprice*0.10) } }
    
                        ~~~ Javascript ~~~ 
                        data: {
                            carprice: 10000
                        }
