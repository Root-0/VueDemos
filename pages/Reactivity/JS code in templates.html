When accessing Vue instance, you can write any valid JS code
as long as it has one expression and does not contain any conditionals or loops (if,for,while etc).

Example:
{ { counter } } can be calculated in place as { { counter += 5 } } or { { counter++ } }

< button v-on:click="increase(5)" >
can be written as
< button v-on:click="counter += 5">

You can also write ternary expressions in vue instance

Example: check if counter greater than 10 and display respective text.
< p > { { counter > 10 ? 'Sorry 10 seats filled' : 'Seats available' } } < / p >
