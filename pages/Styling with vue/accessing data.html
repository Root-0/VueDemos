Unlike in template, where we have access to all data objects, in js code it does not work.
Use this.property to access data properties inside js code.

Copy 
                    
                        data: {
                            name: 'property name here'
                        },
                        methods: {
                            getprop: function(){
                                return this.name;
                            }
                        }
