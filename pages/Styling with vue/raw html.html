You can render raw html with vuejs.
Caution: Use this method of rendering html only when the source is verified, as this can open doors for cross-site-scripting attacks.

Use v-html="property-with-html" to render html inside a tag.

Example:
< h4 v-html="heading-link"> < / h4 > 

                
                data: {
                    heading-link: '< a href="somefile.html">Go to file< / a > '
                }
This renders an < a > tag inside h4 depending on property called 'heading-link'.
