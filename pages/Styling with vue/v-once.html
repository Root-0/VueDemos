All usages of property gets re-rendered when that property value is changed.
i.e. if your property gets updated due to some vue code, all instances of that property on page will be re-rendered to reflect updated values.

You can disable this kind of re-rendering of a property by using v-once on the tag where you use that property.

Example:
< h4 v-once>{ { product.name } }< / a >
This makes h4(product.name) render only once on page load.
