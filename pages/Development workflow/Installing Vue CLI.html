Make sure you have nodejs installed on your machine, as npm is required to install webpack.
We will use the 'webpack-simple' workflow for development.
For advanced use cases like testing, use 'webpack' workflow.

Example: Install vue cli ==> npm install -g vue-cli

Usage: vue init < template-name > < project-name >
Example: vue init webpack my-demo-project

Navigate inside terminal to your folder

[ vue init webpack-simple vue-cli-demo ]
[ cd vue-cli-demo ]
[ npm install ] Install dependencies of webpack
[ npm run dev ] Run server
